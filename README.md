## 🚀 About This Repository

I am not a complete beginner in software development. Before starting Java, I worked with the MERN stack and built full-stack web applications.

Since I was already familiar with programming fundamentals, I decided to begin my Java Backend journey directly with Object-Oriented Programming (OOP) rather than spending time on basic syntax.

This repository serves as a record of everything I learn, build, and practice while transitioning into Java Backend Development.

---

### ✅ Step 2: Project-Based Learning

After completing OOP concepts, I wanted to reinforce my understanding through practical implementation.

As a believer in project-based learning, I built a mini project:

### 👨‍💼 Employee Salary Management System

A console-based application demonstrating:

* Abstraction using an abstract `Employee` class
* Inheritance through different employee types
* Runtime Polymorphism using parent references
* Constructor usage
* Method Overriding

#### Employee Hierarchy

```text
Employee
│
├── Developer
├── Manager
└── Intern
```

Each employee type implements its own salary calculation logic and details display functionality.
